Design Section 1: New Decorators

CSS classes that express visual effects, controlled by CSS custom properties

Effect Types (Semantic)

Patterns declare these functional slots. Recipes map them to decorators.

code_preview

Window chrome, syntax bg, glow

Large metrics/numbers
Text glow, accent color, emphasis
feature_card

Icon backgrounds
Gradient bg, subtle glow

New Decorator Classes

Implemented in each recipe's style CSS, controlled by CSS custom properties.

/* CSS Custom Properties (set by recipe) */
:root {
  --d-glow-radius: 30px;
  --d-glow-intensity: 0.3;
  --d-chrome-bg: linear-gradient(145deg, #1a1a2e, #0d0d14);
  --d-gradient-hint-opacity: 0.08;
}

/* d-glow-primary — Accent glow effect */
.d-glow-primary {
  box-shadow:
    0 0 var(--d-glow-radius) rgba(var(--d-primary-rgb), var(--d-glow-intensity)),
    inset 0 1px 0 rgba(255,255,255,0.08);
  border: 1px solid rgba(var(--d-primary-rgb), 0.4);
}

/* d-terminal-chrome — macOS-style window */
.d-terminal-chrome {
  background: var(--d-chrome-bg);
  border-radius: var(--d-radius-panel);
  border: 1px solid rgba(255,255,255,0.1);
  box-shadow: 0 0 40px rgba(var(--d-accent-rgb), 0.08);
}
.d-terminal-chrome::before {
  /* Traffic light dots */
  content: "● ● ●";
  color: #ff5f57;
}

/* d-stat-glow — Text glow for large numbers */
.d-stat-glow {
  text-shadow: 0 0 var(--d-glow-radius) rgba(var(--d-primary-rgb), 0.4);
}

/* d-gradient-hint-primary — Subtle gradient bg */
.d-gradient-hint-primary {
  background: linear-gradient(
    135deg,
    rgba(var(--d-primary-rgb), var(--d-gradient-hint-opacity)) 0%,
    transparent 50%
  );
}
