Design Section 2: Recipe Schema

New visual_effects config and updated pattern_overrides

New Recipe Field: visual_effects

Top-level config that controls decorator behavior for this recipe.

// recipe-auradecantism.json
{
  "id": "auradecantism",
  "style": "auradecantism",

  // NEW: Visual effects configuration
  "visual_effects": {
    // Master toggle
    "enabled": true,

    // Intensity presets (affects CSS custom properties)
    "intensity": "medium",  // "subtle" | "medium" | "strong"

    // Effect type → decorator mapping
    "type_mapping": {
      "code_preview": ["d-terminal-chrome"],
      "stat_display": ["d-glow-primary", "d-stat-glow"],
      "feature_card": ["d-glass", "d-gradient-hint-primary"],
      "icon_container": ["d-gradient-hint-primary", "d-glow-primary"]
    },

    // Component fallback detection (for patterns without effect_types)
    "component_fallback": {
      "pre": "code_preview",
      "Statistic": "stat_display",
      "Card": "feature_card"
    }
  },

  // Existing fields...
  "decorators": { ... },
  "skeleton": { ... }
}

Intensity Presets

Each recipe sets CSS custom properties based on intensity level.

Subtle
--d-glow-radius: 15px
--d-glow-intensity: 0.15
--d-gradient-hint-opacity: 0.04
Medium (default)
--d-glow-radius: 30px
--d-glow-intensity: 0.3
--d-gradient-hint-opacity: 0.08
Strong
--d-glow-radius: 50px
--d-glow-intensity: 0.5
--d-gradient-hint-opacity: 0.12
