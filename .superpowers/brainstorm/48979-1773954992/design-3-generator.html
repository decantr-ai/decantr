Design Section 3: Pattern & Generator

How patterns declare effect slots and how the generator applies decorators

Pattern effect_types Field (Optional)

Patterns can declare which effect slots they contain. If omitted, generator uses component detection.

// src/registry/patterns/bento-features.json
{
  "id": "bento-features",
  "name": "Bento Features",

  // NEW: Declare effect slots this pattern contains
  "effect_types": {
    "terminal": "code_preview",      // Terminal preview slot
    "stats": "stat_display",         // Stat cards slot
    "features": "icon_container"     // Feature icons slot
  },

  // Existing fields...
  "presets": { ... },
  "default_blend": { ... }
}

Generator Logic Flow

The generator applies decorators automatically during code generation.

1
Load recipe visual_effects config
Check enabled, get intensity, load type_mapping
2
For each pattern in blend, check for effect_types
If declared → use explicit mapping
If not → scan code for components, use component_fallback
3
Map effect types → decorator classes via type_mapping
code_preview → ["d-terminal-chrome"]
stat_display → ["d-glow-primary", "d-stat-glow"]
4
Inject decorators into generated code
Add classes to appropriate elements in pattern output
5
Set CSS custom properties based on intensity
Inject --d-glow-radius, --d-glow-intensity, etc. into page styles

Generated Output Example

WITHOUT visual_effects
function BentoFeatures() {
  return div({ class: css('_grid _gap4') },
    // Plain pre element
    pre({ class: css('_p4 _r2') },
      '$ decantr mcp...'
    ),
    // Plain stat
    span({ class: css('_text3xl _bold') },
      '100+'
    )
  );
}
WITH visual_effects (auradecantism)
function BentoFeatures() {
  return div({ class: css('_grid _gap4') },
    // Terminal with chrome + glow
    div({ class: css('d-terminal-chrome') },
      pre({ class: css('_p4') },
        '$ decantr mcp...'
      )
    ),
    // Stat with glow effects
    span({ class: css('_text3xl _bold d-glow-primary d-stat-glow') },
      '100+'
    )
  );
}
