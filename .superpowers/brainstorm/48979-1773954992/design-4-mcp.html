Design Section 4: MCP Server Updates

Expose visual_effects config to AI assistants for smarter scaffolding

Current MCP Tools

The registry server already exposes 6 tools. We'll extend get_recipe and add one new tool.

search_patterns Find patterns by name/description get_archetype Get full archetype JSON validate_essence Validate essence.json list_components List all components get_recipe Get full recipe JSON ← includes visual_effects now list_archetypes List all archetypes get_visual_effects NEW: Get visual_effects config + available decorators

New Tool: get_visual_effects

Returns visual_effects config for a recipe, plus decorator CSS for reference.

// Tool input
{
  "recipe": "auradecantism"
}

// Tool output
{
  "recipe": "auradecantism",
  "visual_effects": {
    "enabled": true,
    "intensity": "medium",
    "type_mapping": {
      "code_preview": ["d-terminal-chrome"],
      "stat_display": ["d-glow-primary", "d-stat-glow"],
      "feature_card": ["d-glass", "d-gradient-hint-primary"],
      "icon_container": ["d-gradient-hint-primary", "d-glow-primary"]
    },
    "component_fallback": { ... }
  },
  "decorators": {
    "d-terminal-chrome": {
      "description": "macOS-style terminal window with traffic light dots",
      "usage": "Wrap pre/code elements"
    },
    "d-glow-primary": {
      "description": "Box-shadow glow using primary color",
      "usage": "Apply to cards, stats, highlighted elements"
    }
  },
  "intensity_values": {
    "--d-glow-radius": "30px",
    "--d-glow-intensity": "0.3",
    "--d-gradient-hint-opacity": "0.08"
  }
}

AI Usage Example

How Claude/AI uses this when scaffolding:

→
User: "Add a stats section to the dashboard"
→
AI calls get_visual_effects("auradecantism")
Sees: stat_display → ["d-glow-primary", "d-stat-glow"]
→
AI generates code with decorators applied:
span({ class: css('_text3xl _bold d-glow-primary d-stat-glow') }, '100+')
✓
Result: Visually stunning on first scaffold
