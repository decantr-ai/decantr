The problem with AI-generated UI

Every AI code generator races to output faster. None of them think about what happens after they generate.

Without Decantr

Pages don't match each other

Each prompt produces a different layout, spacing, and component style. After 10 iterations, your app looks like it was built by 10 different people.

With Decantr

Consistent design language

Patterns enforce consistent component composition. The Essence spec captures your design intent. Every page matches.

Without Decantr

Every iteration drifts further

AI doesn't remember what it decided before. New features conflict with existing design decisions. Quality degrades over time.

With Decantr

Guard rules prevent drift

Five guard rules validate every change against your spec. Theme, structure, layout, recipe, and density are enforced automatically.

Set up in 30 seconds

Add Decantr to your AI coding assistant. No project changes needed.

Claude Desktop

Add to your claude_desktop_config.json:

{ "mcpServers": { "decantr": { "command": "npx", "args": ["@decantr/mcp-server@beta"] } } }

Claude Code

Add to your settings.json:

{ "mcpServers": { "decantr": { "command": "npx", "args": ["@decantr/mcp-server@beta"] } } }

Cursor

Add to .cursor/mcp.json in your project:

{ "mcpServers": { "decantr": { "command": "npx", "args": ["@decantr/mcp-server@beta"] } } }
