Decantr Schemas

Canonical JSON Schemas for Decantr essence and registry content. These files back the $schema URLs used across official essence documents, patterns, themes, blueprints, archetypes, and shells.

Registry schemas are published from @decantr/registry and essence schemas from @decantr/essence-spec. Execution-pack schemas are published from @decantr/core. Verification report schemas are published from @decantr/verifier.
